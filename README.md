# AI Document Processing Platform

An event-driven, scalable AI document processing platform that ingests, processes, and enables intelligent querying over large collections of documents using OCR, Kafka pipelines, and MCP-based AI agents.

This platform is designed to support **multiple document types** (resumes, legal cases, medical reports, invoices, articles, etc.) and provides both **document-level and collection-level intelligence**.

---

## 🚀 Key Features

- 📄 Upload and process documents of any type (PDF, images, scanned files)
- ⚡ Asynchronous processing using Kafka-based pipelines
- 🔍 OCR-powered text extraction (AWS Textract)
- 🧠 AI-powered summarization, Q&A, and entity extraction
- 🤖 MCP-based AI agent with tool calling over backend APIs
- 📚 Collection-level search and chat across thousands of documents
- 🧩 Entity linking across documents (judges, lawyers, experts, candidates, etc.)
- ⚙️ Redis caching and job state tracking
- 🐘 PostgreSQL-backed structured storage
- 🐳 Fully containerized using Docker & Docker Compose
